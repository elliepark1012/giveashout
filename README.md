### Built With

* React

* Ruby on Rails

* CSS


### Getting Started 

#### Prerequisites


* Install NPM packages

$ npm install

* Next, run the Rails server in one terminal:

$ rails s

* To access the admin page: 

visit: http://localhost:3000/admin

*  Open another terminal and run the React server:
  
$ npm start --prefix client
